# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-10

### Added
- Core prayer times dashboard with real-time status
- Prayer time calculation and progress visualization
- Quran browsing by Surah, Ayah, Juz, Page, Ruku, and Keywords
- Quran search functionality
- Community features page
- Resource library
- Menu/settings page
- Dark mode and light mode with persistent theme storage
- Responsive mobile-first design
- Bottom navigation with 5 main pages
- Animated page transitions
- Geolocation support for accurate prayer times
- Lucide icon integration
- Inter font typography
- Design system with CSS variables (color palette, spacing, typography)
- GitHub Pages deployment
- Comprehensive documentation
- Contributing guidelines
- Code of Conduct
- Security policy

### Technical Details
- Vanilla JavaScript (ES6+) implementation
- Pure CSS with CSS Grid and Flexbox
- Mobile-optimized interface (72px bottom nav, 56px top bar)
- Browser localStorage for preferences and theme
- Event-driven page routing system
- Modular page architecture
- Smooth CSS animations and transitions

### Browser Support
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile, Firefox Mobile)
